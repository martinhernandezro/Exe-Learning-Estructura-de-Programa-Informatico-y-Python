Listas y Diccionarios en Python

Listas en Python

Las listas son uno de los tipos de datos más fundamentales y versátiles en Python. Se utilizan para almacenar una colección ordenada de elementos que pueden ser de cualquier tipo: números, cadenas de texto, otros objetos o incluso otras listas. Son colecciones mutables, lo que significa que sus elementos pueden ser modificados después de ser creados.

Creación de Listas
Las listas se definen utilizando corchetes [] y los elementos se separan por comas.

En este ejemplo, mi_lista contiene solo números enteros, mientras que colores almacena cadenas de texto, y mi_lista_heterogenea tiene una combinación de diferentes tipos de datos.

Acceso a Elementos de una Lista
Los elementos de una lista se acceden mediante su índice, que comienza en 0. También puedes usar índices negativos para acceder a los elementos desde el final de la lista.

Modificación de Listas
Como las listas son mutables, puedes cambiar el valor de uno o más de sus elementos después de haberlas creado.

Operaciones con Listas
Python permite realizar varias operaciones sobre listas, como concatenación, repetición y comprobación de pertenencia.

Eliminar Elementos de una Lista
Puedes eliminar elementos de una lista utilizando métodos como remove(), pop() o del.

Diccionarios en Python

Un diccionario es una estructura de datos no ordenada que se utiliza para almacenar pares de clave-valor. Los diccionarios son útiles cuando necesitamos asociar una clave única con un valor, lo que permite realizar búsquedas rápidas. Al igual que las listas, los diccionarios son mutables, lo que significa que se pueden modificar después de su creación.

Creación de Diccionarios
Los diccionarios se definen utilizando llaves {} y las claves y los valores se separan por dos puntos :. Los elementos del diccionario se separan por comas.

Acceso a Elementos de un Diccionario
Puedes acceder a los valores de un diccionario utilizando sus claves. Si intentas acceder a una clave que no existe, obtendrás un error.

Modificación de Diccionarios
Puedes modificar un valor existente o agregar nuevos pares clave-valor a un diccionario.

Métodos Comunes de Diccionarios
Los diccionarios tienen varios métodos útiles que puedes utilizar para interactuar con ellos.

Comprobación de Pertenencia en Diccionarios
Puedes verificar si una clave está presente en un diccionario usando el operador in.

Iterar sobre Diccionarios
Puedes iterar sobre las claves, valores o ambos del diccionario utilizando un bucle for.

Diferencias entre Listas y Diccionarios

Aunque tanto las listas como los diccionarios son colecciones de datos, existen diferencias clave entre ellos:

-Orden: Las listas son ordenadas, lo que significa que los elementos se almacenan en el orden en que fueron agregados. Los diccionarios, en versiones de Python anteriores a la 3.7, no garantizaban el orden de los elementos. Sin embargo, desde Python 3.7, los diccionarios mantienen el orden de inserción de sus claves.
-Acceso: En una lista, el acceso se realiza mediante índices numéricos, mientras que en un diccionario el acceso se realiza mediante claves, que pueden ser cualquier tipo inmutable (como cadenas, números o tuplas).
-Uso: Las listas se usan principalmente cuando necesitas una colección ordenada de elementos, mientras que los diccionarios son ideales para asociar claves únicas con valores y realizar búsquedas rápidas.
